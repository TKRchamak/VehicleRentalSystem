# VehicleRentalSystem

## Setup

1. run "npm install" // to install all the dependencies,
2. After update codebase need to run "npm run build" to update the build file also.
3. "vercel --prod" this project is deployed in vercel, So deployed this project on the server need to run this command
4. for environment need to visit the "./src/config" to get context.
